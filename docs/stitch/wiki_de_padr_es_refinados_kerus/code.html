Functional Coloring

In the KERUS system, color is not decorative; it is a semantic anchor.

Row A Icons use the signature Obsidian Orange to denote core product destinations and critical navigation nodes.

Row B Icons utilize Neutral Light Grey to recede into the UI, signaling utility actions that are secondary to the user's primary workflow.
