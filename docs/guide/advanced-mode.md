# Modo Avançado

O Modo Avançado desbloqueia campos técnicos extras no editor de fichas técnicas. Ele foi desenvolvido para formuladores que precisam registrar parâmetros físico-químicos detalhados além das proporções básicas.

---

## Como Ativar

Acesse **Configurações** (ícone de engrenagem no menu lateral) e localize a seção **Modo Avançado**. Use o interruptor para ativar.

A alteração é aplicada imediatamente. Abra qualquer ficha técnica no editor para ver os campos extras.

---

## O que é Desbloqueado

### Faixa de pH

Registre os valores mínimo e máximo de pH esperados para a fórmula finalizada.

Útil para:
- Garantir a compatibilidade dos ativos com o pH da formulação
- Definir critérios de aprovação no controle de qualidade
- Documentar a especificação técnica do produto

### Perfil de Viscosidade

Descreva a viscosidade esperada do produto (ex: fluido, gel leve, creme denso).

Útil para:
- Padronizar a textura entre lotes
- Orientar o operador na avaliação visual do produto acabado

### Controle de Versões

Cada alteração significativa na fórmula pode ser registrada como uma nova versão, mantendo o histórico completo de desenvolvimento.

Útil para:
- Rastrear a evolução de protótipos ao longo do tempo
- Manter um histórico para auditorias e laudos de P&D
- Recuperar versões anteriores se necessário

---

## Quando Usar

O Modo Avançado é recomendado para:

- Laboratórios cosméticos com necessidade de documentação regulatória
- Formuladores que entregam dossiês técnicos completos aos clientes
- Equipes de P&D que iterem sobre protótipos e precisam de rastreabilidade

Para uso cotidiano, o modo padrão oferece todos os campos necessários para criação e exportação de fichas profissionais.

::: warning Desativar o Modo Avançado não apaga os dados já registrados
Se você desativar o modo, os campos extras ficam ocultos, mas as informações salvas são preservadas. Reativar o modo faz os campos voltarem a aparecer com os dados intactos.
:::
