# Aparência e Tema

O Kerus oferece dois temas visuais que podem ser alternados a qualquer momento, sem reiniciar o sistema.

---

## Tema Escuro e Tema Claro

**Tema Escuro** — interface com fundo escuro e texto claro. Indicado para uso prolongado em ambientes com pouca luz ou em monitores de alta luminosidade.

**Tema Claro** — interface com fundo claro e texto escuro. Indicado para ambientes bem iluminados ou preferência pessoal por interfaces mais tradicionais.

A cor de destaque laranja do Kerus é mantida em ambos os temas.

---

## Como Alterar o Tema

Há duas formas de alternar o tema:

**Pelo menu lateral**
: Clique no ícone de sol/lua na parte inferior do menu lateral. O tema muda imediatamente.

**Pelas Configurações**
: Acesse **Configurações** no menu lateral, localize a seção **Aparência** e clique no botão de alternância.

A preferência é salva automaticamente e persiste entre sessões.
