# Gerenciamento de Fórmulas

As fórmulas são o coração do Kerus. Elas foram projetadas para oferecer o rigor técnico exigido por formuladores profissionais.

## Estrutura Técnica
Diferente de uma receita comum, uma fórmula Kerus é dividida em **Fases**:
- **Fase A, B, C...**: Organize a ordem de adição e processamento.
- **Precisão Decimal**: Suporte a pesos e porcentagens com alta precisão.
- **Dados Técnicos**: Campos específicos para **Nomenclatura INCI** e função de cada matéria-prima.

## Funcionalidades de Elite
- **Editor Assistido por IA**: Use o poder do Gemini para converter uma foto de um rótulo ou uma lista de ingredientes em texto em uma ficha estruturada instantaneamente.
- **Consistência de Peso**: O sistema alerta automaticamente se sua fórmula não fechar em 100% ou se houver inconsistências de peso.
- **Status de Ciclo de Vida**: Mantenha rascunhos em progresso e marque como **FINAL** apenas quando a fórmula estiver validada para produção.
