# Atualizações e Melhorias

Acompanhe aqui o histórico de evoluções, correções e novas funcionalidades do Kerus.

## [alpha-1.0.2] — 2026-03-25

### 🚀 Novidades e Branding
- **Renomeação Oficial**: O projeto agora se chama oficialmente **Kerus**. Todas as referências a "formula-generator" e "Ficha Técnica Pro" foram removidas para unificar a marca.
- **Nicho de Mercado**: Documentação atualizada para focar explicitamente nos setores de **Cosméticos** e **Gastronomia Profissional**.

### 🎨 Interface e Experiência (UX/UI)
- **Refinamento de Gradients**: O gradiente dos cards no tema claro foi ajustado para ser mais sutil. Agora os cards apresentam um branco mais "vivid" e limpo, mantendo a profundidade técnica sem poluição visual.
- **Saturação e Contraste**: Melhoria na saturação das cores dos cards no Hub (Formulação, Orçamento, Dashboard) para garantir um visual "premium" e vibrante em ambos os temas.
- **Correção de Scroll**: Resolvido o problema que impedia a rolagem vertical em algumas telas após a última atualização de design.

### 📚 Documentação
- **Nova Seção de Atualizações**: Criação desta página para registro transparente de mudanças.
- **Melhoria dos Guias**: Os guias de Fórmulas e Orçamentos foram reescritos em português profissional, com foco em precisão industrial.

## [alpha-1.0.1] — 2026-03-24

### 🎨 Design System
- Introdução do sistema **Obsidian Architect**.
- Implementação de temas dinâmicos (Light/Dark) baseados em tokens de design.
- Padronização de componentes interativos (Botões, Inputs, Selects).

### ⚙️ Funcionalidades Base
- Editor de Fórmulas com suporte a fases.
- Sistema de Orçamentos vinculado a dados técnicos.
- Integração com IA Gemini para extração de ingredientes.
