# Primeiros Passos

Bem-vindo ao Kerus. Este guia mostra o fluxo essencial do sistema para você começar a trabalhar em minutos.

---

## Visão Geral do Fluxo

O Kerus organiza seu trabalho em três etapas naturais:

1. **Configurar** — preencha os dados da sua empresa nas Configurações para que apareçam nas exportações.
2. **Formular** — crie Fichas Técnicas com ingredientes, fases e proporções.
3. **Comercializar** — gere Orçamentos a partir das fichas, com custos calculados automaticamente.

---

## Passo 1: Configure seu Perfil

Antes de criar qualquer ficha, acesse **Configurações** (ícone de engrenagem no menu lateral) e preencha:

- Nome da empresa ou laboratório
- Nome do responsável técnico
- Telefone de contato

Esses dados aparecem no cabeçalho de todas as fichas e orçamentos exportados.

::: tip Faça isso uma única vez
Após salvar, as informações são aplicadas automaticamente em todos os documentos futuros. Você não precisa preencher novamente a cada exportação.
:::

---

## Passo 2: Crie sua Primeira Ficha Técnica

No menu lateral, acesse **Fichas Técnicas** e clique no botão **Nova Ficha**.

O editor será aberto. Preencha:

1. **Nome da fórmula** — identifique o produto (ex: "Creme Hidratante Base")
2. **Tamanho do lote** — quantidade total que será produzida (ex: 1000g)
3. **Fase** — organize os ingredientes por etapa de adição (ex: Fase A, Fase B)
4. **Ingredientes** — adicione cada matéria-prima com seu percentual e nome INCI, se aplicável

O sistema valida em tempo real se a soma dos percentuais é exatamente 100%. Enquanto a fórmula estiver sendo desenvolvida, mantenha o status como **Rascunho**.

Quando a fórmula estiver pronta para produção, mude o status para **Final**.

::: info Apenas fichas no status Final podem ser exportadas como PDF
Isso garante que documentos formais reflitam apenas fórmulas validadas.
:::

---

## Passo 3: Gere um Orçamento

Com uma ficha salva, acesse **Orçamentos** no menu lateral e clique em **Novo Orçamento**.

1. Vincule a ficha técnica desejada
2. Ajuste o tamanho do lote para o volume cotado
3. Defina a margem de lucro e os custos adicionais
4. Revise o custo final e o preço de venda sugerido

O orçamento fica salvo com status **Pendente** até que o cliente responda. Atualize para **Aprovado** ou **Arquivado** conforme o andamento.

---

## Onde Encontrar Ajuda

- **Botão `?`** — visível ao lado dos campos quando o Modo Ajuda está ativado. Clique para ver uma explicação rápida sem sair da tela.
- **Esta documentação** — use a barra lateral para navegar pelos guias de cada módulo.
- **Modo Avançado** — se precisar de campos extras como pH, viscosidade e versionamento de fórmulas, ative nas Configurações.
