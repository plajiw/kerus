# Orçamentos

Gere propostas comerciais profissionais a partir das suas fórmulas.

## O que é um Orçamento?
O Orçamento é a ponte entre sua formulação técnica e a entrega para o cliente final, permitindo gerenciar margens, impostos e fretes.

## Funcionalidades
- **Vínculo com Fórmulas**: Utilize dados técnicos para gerar preços.
- **Dados do Cliente**: Gerencie informações de contato e entrega.
- **Exportação Profissional**: Gere PDFs prontos para envio.
